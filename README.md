# Zomato Power BI Dashboard

## Project Overview
This project is an interactive Power BI dashboard built using Zomato restaurant data to analyze customer behavior, restaurant performance, ratings, cuisines, and delivery insights.

## Features
- Restaurant ratings analysis
- City-wise restaurant distribution
- Online delivery analysis
- Cuisine popularity insights
- Cost and pricing analysis
- Interactive visual filters and KPIs

## Tools & Technologies
- Power BI
- Data Visualization
- Data Cleaning
- Excel / CSV Dataset

## Dashboard Insights
- Identified top-performing restaurants
- Compared online vs offline delivery trends
- Analyzed customer ratings across cities
- Visualized cuisine demand patterns
